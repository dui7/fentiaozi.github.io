# docker中redis导出数据

# 查看redis文件存放目录位置

1.进入redis-cli或用rdm工具进入console 2.查看redis文件存放目录位置

CONFIG GET dir
1

local:0>CONFIG GET dir

  1. dir
  2. /data

我的是在/data目录

# 导出数据

1.进入redis-cli或用rdm工具进入console 2.查看redis文件存放目录位置 执行导出命令，会在/data目录下生成一个dump.rdb

save
1

# 导入数据

1.进入redis-cli或用rdm工具进入console 2.查看redis文件存放目录位置 我的是/data 3.停止redis容器
4.把dump.rdb 通过docker cp 复制到需要导入的redis容器的/data目录下

docker cp dump.rdb redis:/data/
1

4.启动容器

docker start redis
1
